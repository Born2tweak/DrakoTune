# Research Prompt Registry

Each prompt is bounded by a decision and a stopping condition. Research updates evidence; it does not authorize adoption.

| ID | Domain | Decision supported |
|---|---|---|
| A | Product and user evidence | Initial promise, workflow, and launch strata |
| B | Code and architecture | What can safely evolve versus be replaced |
| C | Audio quality and listening standards | Applicable measurement/listening designs |
| D | Statistical validity | Independent units, models, uncertainty, power |
| E | Dataset and provenance | What may be used for which purpose |
| F | Automatic mixing research | Champion/challenger candidates and gates |
| G | Enhancement and assessment models | Domain/license readiness of candidate tools |
| H | Professional listening recruitment | Panel, qualification, compensation, contracts |
| I | Desktop UX and project model | Smallest local-first coherent product |
| J | Packaging and licensing | Distribution branch and reproducible builds |
| K | Performance | Feasible duration/hardware/streaming targets |
| L | Security, privacy, and claims | Data minimization and claim governance |
| M | Continuous research | Safe source monitoring and roadmap renewal |
