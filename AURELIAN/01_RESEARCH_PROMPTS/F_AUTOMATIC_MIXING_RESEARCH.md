# F — Automatic Mixing Research

**Decision:** determine whether a current method merits an offline challenger experiment.

Review primary papers and official code for automatic mixing, differentiable DSP, effect-chain parameter estimation, and style transfer. Extract task, input assumptions, reference requirement, dataset/rights, model/code license, evaluation design, compute, latency, interpretability, failure modes, and reproducibility. Compare directly with DrakoTune’s isolated lead-vocal, local-first constraints.

**Exclusions:** adoption based on paper metrics, demos, or permissive code when weights/data are restricted.

**Deliverable:** readiness cards with reproduce/reject/defer decision and falsifiable experiment.

**Stop when:** each candidate has a rights-safe replication path and explicit champion-comparison gate, or is rejected.
