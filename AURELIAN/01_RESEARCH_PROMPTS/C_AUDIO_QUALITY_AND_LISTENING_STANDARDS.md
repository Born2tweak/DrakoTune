# C — Audio Quality and Listening Standards

**Decision:** choose valid evaluation methods by task and impairment size.

Read current ITU/EBU standards and implementation guidance for loudness, true peak, small-impairment listening, MUSHRA, and full-reference perceptual models. Record required references, anchors, listener training, environment, randomization, and reporting. Test whether each method applies to dry-to-wet creative vocal processing, cleanup, or only codec/device impairment.

**Exclusions:** calling a custom browser comparison “MUSHRA” without required design elements; using loudness or PEAQ as an artistic verdict.

**Deliverable:** task-to-method applicability matrix and prohibited interpretations.

**Stop when:** every quality claim has a valid method or is explicitly unmeasurable with current evidence.
